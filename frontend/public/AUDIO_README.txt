To add the echo ping sound effect:

1. Find or create a short "echo/sonar ping" sound (mp3 or wav format, 1-2 seconds)
2. Name it "echo-ping.mp3" or "echo-ping.wav"
3. Place it in this public folder
4. The logo hover will play the sound automatically

For a quick test, you can use any short audio file or create one using online tools.

Example: Search for "sonar ping sound effect" and download a free one.
